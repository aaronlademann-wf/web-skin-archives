account_licenses ¶

GET https://wf-integration-apis.appspot.com/api/v0.1/account_licenses/ ¶

Deprecated since version 0.3.

All account_licenses resources.

Example request : 

GET https://wf-integration-apis.appspot.com/api/v0.1/account_licenses/ HTTP/1.1
Host: workiva.appspot.com
Accept: application/json

Example response : 

HTTP/1.1 200 OK
Vary: Accept
Content-Type: application/json

{
  "messages": [],
  "reason": "OK",
  "results": [
    {
      "url": "https://wf-integration-apis.appspot.com/api/v0.1/account_licenses/QWNjb3VudExpY2Vuc2UeQWNjb3VudExpY2Vuc2U6MjAwMQ/"
    },
    {
      "url": "https://wf-integration-apis.appspot.com/api/v0.1/account_licenses/QWNjb3VudExpY2Vuc2UeQWNjb3VudExpY2Vuc2U6MjMxMzQ/"
    }
  ],
  "status": 200
}
Query Parameters:
 
  • cursor – cursor. default is None
  • limit – number of results to return. default is None (i.e., no limit; return all available results)
  • verbose – true or 1 if including all the available properties of the resource in the response; false or 0 if including only the URL to the resource in the response. default is false
Status Codes:
  • 200 – no error
  • 400 – invalid request
  • 401 – authentication error
  • 403 – permission error
  • 403 – authentication error
