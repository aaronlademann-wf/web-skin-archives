Dropdown menus

Toggleable, contextual menu for displaying lists of links. Made interactive with the dropdown JavaScript plugin.

Example

Wrap the dropdown's trigger and the dropdown menu within .dropdown, or another element that declares position: relative;. Then add the menu's HTML.

Button groups

Group a series of buttons together on a single line with the button group. Add on optional JavaScript radio and checkbox style behavior with our buttons plugin.

1 <div class="btn-group">

"Indeterminate"

Add .indeterminate to the progress elem to make the bar go 100% width to indicate an "indeterminate" amount of time remaining.

You can also use a .progress-label element to keep your users informed in the case there is an indeterminate state involved.

Modern browsers achieve this effect using CSS animations and striped gradients. Older browsers (like everyone's favorite pal Internet Explorer) use a fallback animated .gif.

1 <div class="progress indeterminate">
2     <div class="progress-bar"></div>
3     <span class="progress-label">Processing&hellip;</span>
4 </div>
